# diffiew

A minimalistic diff viewer built with React and TypeScript.

## Features

- Side-by-side diff comparison
- Character-level diff highlighting
- Edit and view modes
- Dark mode by default
- File loading support

## Getting Started

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```

## Usage

1. Paste or load files into the left and right panels
2. Toggle between editing and viewing modes
3. Use the swap button to exchange left and right content

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Patience Diff algorithm
